# Metal Madera Home – Sitio Web de Catálogo

https://metalmaderahome.com/

Sitio web de catálogo corporativo desarrollado para Metal Madera Home, una empresa dedicada a la fabricación y comercialización de muebles y productos para el hogar.

La plataforma presenta de forma clara los productos, la identidad de la marca y la información relevante para clientes, ofreciendo una experiencia visual limpia y una navegación intuitiva. El diseño está orientado a resaltar los materiales, acabados y estilos de los muebles, adaptándose a distintos dispositivos.

Características principales:

- Sitio web de catálogo de productos
- Diseño responsivo
- Presentación visual de muebles y productos para el hogar
- Estructura clara y navegación sencilla
- Optimización básica para visibilidad en buscadores (SEO)

Objetivo del proyecto:
Brindar a la empresa una vitrina digital profesional que fortalezca su presencia en internet y facilite la exhibición de su portafolio de productos.
